## 1. Что такое проект

**ОРЁЛ** — лендинг-страница B2B SaaS продукта: ИИ-ассистент для анализа звонков отдела продаж.  
Целевая аудитория: **РОПы** (Руководители отделов продаж) и **Собственники бизнеса** на российском рынке.  
Главная цель страницы: **лидогенерация** — сбор заявок на демо.

**Создатель продукта**: Кирилл Соседов (`k-sosedov@yandex.ru`, Telegram `@AiConsult3`)  
**Блог**: https://teletype.in/@aiconsult/zr7XkZa3KOu

> **Для новой сессии**: Наиболее критичные проблемы — P0.1 (токен безопасности) и P0.2 (CTA скрыта на мобайле). Начинать с них.

## 4. Архитектура лендинга — секции по порядку

### 4.1 Header (sticky, z-50)
- Лого + текст "ОРЁЛ" / "Оцифровка речевых лидов"

### 4.2 Hero Section
- Тёмно-синий градиент `#1e293b → #0f172a` + `<NeuralBackground />`

- Подпись: "Видео разбора + практический гайд. Бесплатно."
- 3 стат-карточки: **+28% к прибыли**, **8 часов** экономии/нед, **100%** звонков

### 4.3 Problems Section (bg-gray-50)

## 5. LeadForm — логика

**Поля**: Имя (обязательно), Телефон +7 (обязательно, 10 цифр), Способ связи (Telegram/WhatsApp/Звонок), Согласие на обработку ПД

**Сабмит**: POST `/api/submit-form` → Flask → Telegram Bot + `submissions.json`

**После успеха**: Redirect на `/thank-you?name=...&method=...`

**Thank You Page**: Кнопки → YouTube демо-видео + Teletype гайд, сообщение об обратном звонке

**⚠️ P0 БЕЗОПАСНОСТЬ**: Telegram Bot Token + Chat ID захардкожены в `app.py` (строка 12) — **живая уязвимость**. Любой, кто видит репозиторий, может рассылать спам или перехватить лиды. Переместить в `.env` немедленно.

## 7. Компонентная система

### Reveal.jsx — анимации при скролле
- `FadeUp` — плавный подъём (Framer Motion, `whileInView`)
- `FadeIn` — плавное появление
- `StaggerContainer` — контейнер с задержками дочерних элементов
- `StaggerItem` — дочерний элемент Stagger (задержка 80ms между)
- `ScrollImage` — изображение ч/б → цвет при скролле (используется для фото Кирилла)

### AnimatedNumber.jsx
- Анимированный счётчик (IntersectionObserver)
- Props: `value`, `duration=2000`, `suffix`, `prefix`
- ⚠️ Применяет класс `stat-number` сам + родитель тоже применяет — двойное применение
